# 🏀 Sports App - Angular + Node.js

🚧 Proyecto en desarrollo 🚧

---

## 📌 Descripción del Proyecto

Aplicación web fullstack para la gestión de jugadores, equipos y partidos de baloncesto.
Permite a los usuarios registrarse, iniciar sesión y gestionar sus favoritos, además de administrar información de equipos y partidos.

---

## ⚙️ Funcionalidades

*  Autenticación con JWT (login y registro)
*  CRUD de jugadores
*  CRUD de equipos
*  CRUD de partidos
*  Sistema de favoritos por usuario
*  Protección de rutas (Auth Guard + Middleware)

---

## 🖥️ Tecnologías utilizadas

### Frontend

* Angular
* TypeScript
* HTML / CSS

### Backend

* Node.js
* Express
* MySQL

### Otros

* JWT (autenticación)

---

## 📁 Estructura del proyecto

```
/frontend   → Aplicación Angular
/backend    → API REST con Express
```

---

## 🚀 Cómo ejecutar el proyecto

### 1. Clonar repositorio

```
git clone https://github.com/DanielMena4/Core-Ingenier-a-Web.git
```

---

### 2. Backend

```
cd backend
npm install
npm run dev
```

---

### 3. Frontend

```
cd frontend
npm install
ng serve
```
